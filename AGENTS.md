## Jules: Project Context
This is a module for a B2B Deathcare platform.

### Critical Context:
- **Tone:** Compassionate, secure, and professional.
- **Architecture:** We use "Vibe Coding" principles—prioritize rapid UI iteration using Cursor and Claude Code.
- **Security:** RLS (Row Level Security) is mandatory. We never expose family media without explicit permission.
- **Visuals:** The UI must feel "premium" and "slick" to justify the k setup fee for funeral homes.

### Tech Constraints:
- Use `lucide-react` for icons.
- Use `shadcn/ui` for standard dashboard components.
- Use `three.js` specifically for the 3D Wall component.

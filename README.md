# Gallery-360: Multimedia Memorial Module

## Overview
An immersive, WebGL-powered gallery designed for the Digital Life Closeout platform. It uses a 360-degree "3D Wall" UI to curate and showcase the shared memories of a deceased loved one.

## Tech Stack
- **Framework:** Next.js (App Router)
- **3D Graphics:** Three.js / React-Three-Fiber
- **Backend:** Supabase (Auth, RLS, Storage)
- **AI/Metadata:** Immich API & AWS Rekognition
- **State:** Zustand

## Core Features
- **Kinetic 3D Wall:** Infinite, curved media scrolling with 3D tilting effects.
- **Shared Space:** Collaborative uploads with tiered visibility (Public/Family/Admin).
- **AI Augmented:** Facial grouping and automatic content scrubbing.
- **Story Integration:** Text and voice note "tags" for media items.
